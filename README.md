## 一、先聊搬瓦工 VPS：它到底是个什么东西

搬瓦工的英文叫 BandwagonHost，背后是加拿大的 IT7 公司，2004 年成立，算得上海外 VPS 圈子里"老字号"级别的存在。它最早是靠便宜 KVM VPS 起家的，后来慢慢把重点挪到了为中国大陆用户优化线路上——尤其是 CN2 GT、CN2 GIA 这类电信高端回程线路，在晚高峰也能保持相对稳定的连接，这一点在海外 VPS 市场里确实不算多见。

- **线路优化做得早、做得稳**：CN2 GIA-E 套餐可选机房多，三网回程质量在线，建站、科学上网、跨境访问都比较顺手。

- **优惠码可循环使用**：买的时候用的码，续费、升级的时候自动沿用同折扣，这点挺良心。

不过它也有几个不太能回避的短板，咱们后面会展开讲。
